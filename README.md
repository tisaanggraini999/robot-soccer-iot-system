# Vision-Based Robot Soccer Control System
## TCP/IP • Redis • Computer Vision • Real-Time Control

This project implements a **distributed robotic soccer system** that integrates
computer vision, networking, and real-time embedded motor control.

## End-to-End Pipeline

Camera → OpenCV Vision → Redis IPC → Control Logic → Serial → Robot Controller

## Key Features

- Real-time ball detection using OpenCV
- Angle & distance estimation
- Redis-based inter-process communication
- TCP/IP-ready architecture
- Serial-based embedded motor control
- Modular & scalable system design

## Technologies

- Python (OpenCV, Redis, TCP/IP)
- ESP32 / Arduino
- Serial Communication
- Distributed System Architecture

## Use Cases

- Robot soccer competitions
- Vision-based autonomous robots
- Research & robotics education
- Multi-robot control systems
